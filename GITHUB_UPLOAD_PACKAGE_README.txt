## 🚀 Quick Upload Steps

### Step 1: Create GitHub Repository
```
1. Go to https://github.com/new
2. Name: julius-baer-simulators
3. Visibility: Public
4. Create repository
```

### Step 2: Upload Files
**Easiest method - GitHub Web UI:**
```
1. Click "Add file" → "Upload files"
2. Drag-and-drop all 6 files
3. Commit message: "Initial simulators with 2 strategic planning tools"
```

**Command line method:**
```bash
git clone https://github.com/YOUR-USERNAME/julius-baer-simulators.git
cd julius-baer-simulators
# Copy all 6 files here
git add .
git commit -m "Initial simulators: Data Integration + Cloud/FinOps with disclaimers"
git push origin main
```

### Step 3: Enable GitHub Pages
```
1. Settings → Pages
2. Source: "Deploy from a branch"
3. Branch: main, Folder: root
4. Save
```

**Your site goes live at:** `https://YOUR-USERNAME.github.io/julius-baer-simulators`
